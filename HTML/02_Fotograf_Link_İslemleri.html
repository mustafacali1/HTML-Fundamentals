Link İşlemleri

Link vermek ve kullanıcıyı ilgili linke yönlendirmek için kllanılan 'a' elemenditidir. Temelde 2 ana özelliği bulunmatadır.
1-) 'href' => Kullanıcıya linke hangi web sayfasına gidecekse ilgili web sayfasının linkini veriyoruz.
2-) 'target' => Yönlendirilen sayfanın nasıl açılacağını belirler. Yani linke tıklandığında bulunan sayfada mı yoksa yeni sayfada mı açılsın kararını burada veriyoruz.

Fotoğraf işlemleri

Sayfa üzeinde bir fotoğraf görüntelemek için 'img' elementini kullanıyoruz. Temel anlmada 3 tane özelliği bulunmaktadır.
1-)src: Source anlamına gelmektedir. Fotoğrafın kaynak yolunu belirtir.
2-)alt: Alternatif metin olarak kullanılmaktadır. Fotoğraf yolunun bulmazsa bunun ile verdiğimiz bilgi ekrana yazılır..
3-)title: Fotoğraf başlığı olarak kullanılır. Mouse ile üzerine geldiğimizde bize gösterilen ifadedir.
